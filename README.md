# Travel Management System

A PHP & MySQL based Travel Management System styled with Bootstrap.

## Developer Info
- **Author:** Vijay Mahes
- **Email:** [Vijaypradhap2004@gmail.com](mailto:Vijaypradhap2004@gmail.com)
- **GitHub Repository:** [Travel-Management-System-php](https://github.com/vijaymahes9080/Travel-Management-System-php.git)

## Features
- Interactive slides and packages
- Category and subcategory filtering
- Travel booking and enquiries
- Comprehensive Admin Dashboard to manage packages, categories, enquiries, and subcategories

## Tech Stack
- PHP (Backend)
- MySQL (Database)
- HTML5, CSS3, Javascript, Bootstrap (Frontend)
